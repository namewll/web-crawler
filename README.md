# web-crawler
web_crawler Python 爬虫与加密技术学习项目  技术栈 Python 3.x 、多线程 、异步爬虫、正则 、XPath、AJAX、Selenium 、Base64、MD5、AES、RSA 加密 、CSV 数据存储 、JS 逆向 、YOLO 验证码识别  项目内容 基础爬虫：多线程、正则、XPath、AJAX 动态数据、Selenium 自动化 加密实践：Base64/MD5/AES/RSA 加解密，逆向爬取多重加密网站 实战项目：猎上网招聘数据、民宿评论、影视短评等多场景数据采集 辅助能力：JS 基础与逆向、YOLO 验证码识别 亮点 体系化覆盖爬虫核心技术与主流加密算法，从基础到实战完整链路，代码模块化可复用，适合作为爬虫与数据安全方向的学习成果展示。
